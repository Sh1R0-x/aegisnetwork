Comment intégrer dans Outlook

  1. Ouvrir ce fichier HTML dans un navigateur (Chrome recommandé).
  2. Sélectionner le contenu de la signature souhaitée (version claire ou sombre, dans le cadre de preview).
  3. Copier (Ctrl+C) le contenu sélectionné.
  4. Dans Outlook : Fichier → Options → Courrier → Signatures.
  5. Créer une nouvelle signature (ex: "Aegis Network").
  6. Coller (Ctrl+V) dans l'éditeur de signature.
  7. Vérifier que le rendu est correct et que les liens fonctionnent.
  8. Définir comme signature par défaut pour les nouveaux messages et les réponses.

Pour Gmail / Google Workspace

  1. Même méthode : copier depuis le navigateur, coller dans Gmail → Paramètres → Signature.
  2. Gmail peut légèrement modifier le rendu. La version compacte (sans baseline) fonctionne mieux dans Gmail.
  3. Utiliser la version claire dans la majorité des cas (Gmail impose un fond blanc).

Quelle version choisir ?

• Fond clair — À utiliser par défaut pour Outlook, Gmail, et la plupart des clients mail. Le fond blanc est imposé par le client dans 99% des cas.

• Fond sombre — À utiliser si votre client mail supporte un thème sombre natif (Outlook dark mode, Thunderbird dark). La plupart des clients mail en mode sombre inversent les couleurs automatiquement, mais pour les cas où le fond est réellement sombre, cette version garantit un rendu optimal.

Notes techniques

• Le logo utilise un SVG inline avec fallback VML (lettre "A") pour Outlook desktop.

• Pour un rendu encore plus fiable, vous pouvez remplacer le SVG par une image PNG 52×52px hébergée sur aegisnetwork.fr.

• Les styles sont 100% inline pour la compatibilité Outlook.

• Aucune image externe, pas de CSS externe, pas de JavaScript.

• Police : Arial (web-safe) est utilisée à la place d'Inter pour la compatibilité email.

• Largeur max : 520px pour un rendu optimal sur tous les écrans.

• Le séparateur (border-left: 3px solid) reprend l'accent bleu de la carte de visite pour la cohérence.

• Icônes : SVG inline Lucide (phone, mail, globe, map-pin) en stroke 2px, cohérentes avec la charte.
