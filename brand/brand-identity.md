## 01. Fondations de marque

### Résumé

Aegis Network accompagne les TPE et PME dans la gestion, la sécurisation et l'optimisation de leur infrastructure informatique. L'entreprise simplifie la connectivité professionnelle — internet haut débit, téléphonie IP, infrastructure réseau — pour que les dirigeants puissent se concentrer sur leur activité.

### Positionnement

Partenaire technique de proximité, spécialisé en connectivité et téléphonie d'entreprise. Ni revendeur généraliste, ni opérateur impersonnel : un expert local qui comprend les enjeux des petites structures.

### Promesse

Une connectivité fiable, optimisée et maîtrisée, accompagnée par un interlocuteur humain et réactif. Pas de jargon inutile, pas de promesses vides : des résultats concrets et mesurables.

### Valeurs

| Valeur | Signification |
|---|---|
| **Fiabilité** | Ce qui est promis est livré |
| **Clarté** | Pas de zone d'ombre technique |
| **Accompagnement** | Un partenaire, pas un fournisseur |
| **Performance** | Chaque milliseconde compte |

### Ton de communication

- Professionnel mais accessible — pas de jargon
- Rassurant sans être paternaliste
- Concret, orienté résultats
- Précis et factuel
- Humain, pas robotique

**Structure des messages :**
1. Le bénéfice pour le dirigeant (quoi)
2. Ce que ça résout concrètement (pourquoi)
3. Comment on le fait (en 1 phrase, facultatif)

### Vocabulaire

**Privilégié :** accompagner, optimiser, concret, fiable, maîtrisé

**Interdit :** leader, n°1, meilleur, solution clé en main, disruption, révolutionnaire, « cyber » sans contexte

### Mots-clés de direction artistique

Fiabilité · Clarté · Performance · Premium · Technologique · Connectivité · Sobriété · Confiance

### Do / Don't

| ✓ Do | ✗ Don't |
|---|---|
| Visuels de data centers, fibres, réseaux réels | Néons verts, ambiance "matrice" ou "hacking" |
| Tons bleus profonds, espaces aérés | Trop de violet dominant |
| Glassmorphisme subtil | Icônes remplies ou incohérentes |
| Icônes Lucide React (trait fin, cohérent) | Fonds très saturés ou trop contrastés |
| Photos professionnelles, lumière maîtrisée | Textures lourdes, motifs agressifs |
| Dégradé bleu → violet discret | Superposition d'effets (glow + gradient + shadow) |

## 02. Système de logo

### Description

Le logo Aegis Network est un « A » stylisé en forme de bouclier/chevron, évoquant la protection (Aegis) et la connectivité (nodes fibre). Il fonctionne en lockup horizontal avec le wordmark, ou seul en tant qu'icône.

### Versions disponibles

| Version | Fichier | Usage |
|---|---|---|
| Icon gradient | `assets/logo-icon-gradient.svg` | Fond sombre (digital) |
| Icon bleu | `assets/logo-icon-blue.svg` | Fond clair ou sombre, monochrome bleu |
| Icon blanc | `assets/logo-icon-white.svg` | Fonds colorés, overlay photos |
| Icon sombre | `assets/logo-icon-dark.svg` | Fond clair, impression |
| Icon épuré | `assets/logo-icon-clean.svg` | Petites tailles (< 32px), favicon |

### Lockup (icône + wordmark)

- Wordmark : `AEGIS NETWORK` — Inter Black (900), letter-spacing: 0.06em; word-spacing: 0.2em
- "NETWORK" en gradient (blue → violet) sur écran, ou en Aegis Blue en print
- Baseline : "Conseil & Optimisation IT" — Inter Bold, 0.25em tracking, uppercase

### Marges de protection

Espace libre minimum = hauteur du logo ÷ 4, sur tous les côtés.

### Tailles minimales

| Version | Digital | Print |
|---|---|---|
| Lockup complet | 160px largeur | 35mm |
| Icône avec nodes | 32px | 10mm |
| Icône épurée | 16px | 5mm |

### Usages interdits

- Ne pas déformer (étirement, compression, rotation)
- Ne pas recoloriser hors palette
- Ne pas placer sur fond photographique chargé
- Ne pas ajouter d'effets (drop-shadow, bevel, 3D)
- Ne pas modifier le gradient
- Ne pas séparer icône et wordmark dans le lockup

## 05. Univers graphique

### Iconographie

- **Bibliothèque** : Lucide React
- **Style** : stroke uniquement, épaisseur 2px, caps arrondis
- **Taille par défaut** : 24px
- **Couleur** : Optical Blue (#3B82F6) ou Deep Violet (#7C3AED)

### Glassmorphisme (glass-card)

```css
background: rgba(15, 23, 42, 0.6);
backdrop-filter: blur(12px);
border: 1px solid rgba(255, 255, 255, 0.1);
border-radius: 24px;
box-shadow: 0 25px 50px -12px rgba(0, 0, 0, 0.5);
```

### Boutons

| Type | Fond | Bordure | Texte |
|---|---|---|---|
| Primaire | Gradient blue-600 → violet-600 | — | Blanc, bold |
| Secondaire | rgba(255,255,255,0.05) | 1px white/10 | Blanc, bold |
| Outline | Transparent | 1px Optical Blue | Optical Blue |

Border-radius : 12px · Hauteur : 48px (standard), 56-64px (hero CTA)

### Effets visuels

| Effet | Usage |
|---|---|
| `.premium-glow` | Box-shadow bleu sur images/cards en avant |
| `.glow-button` | Box-shadow + hover scale sur CTAs |
| `.fiber-beam` | Lignes 1px animées en arrière-plan |
| `.animate-float` | Animation lente haut/bas (hero image) |
| `.animate-hero-glow-*` | Orbes gradient en dérive lente |
| `.hero-grid` | Grille technique subtile (opacity 2%) |

Tous respectent `prefers-reduced-motion`.

### Traitement photos

- Opacité réduite (60-80%)
- mix-blend-mode: lighten ou screen
- Overlay gradient (from-background-deep)
- Overlay coloré (blue-600/20 ou violet/20)
- `loading="lazy"`, `alt` descriptif en français

### Principes de formes

- Border-radius : 20-24px (cards), 12px (boutons), 100px (pills)
- Pas d'angles vifs sauf séparateurs linéaires
- Dégradés : toujours Aegis Blue → Deep Violet, jamais l'inverse
- Séparateurs : border-white/5 (subtils)

## 06. Applications de marque

### Site web

One-page : Navbar → Hero → Stats → Solutions → VoIP → Infrastructure → CTA → Contact → Footer

### Carte de visite

- Format : 85 × 55 mm, bleed 3mm, safe zone 5mm
- Recto : fond clair (Slate 50), accent bar gradient à gauche, info contact
- Verso : fond sombre (Slate 900), logo centré, baseline, URL
- Fichier : `business-card.html`

### Signature email

- HTML compatible Outlook/Gmail/Apple Mail
- Table-based, inline styles, Arial (fallback web-safe)
- Séparateur vertical bleu Aegis
- Fichier : `email-signature.html`

### Documents commerciaux

- En-tête : lockup logo + baseline, aligné gauche, fond blanc
- Accent : filets et titres en Aegis Blue
- Texte : Slate 900 sur blanc, Inter Regular 10-11pt
- Pas de fond sombre sur documents papier
